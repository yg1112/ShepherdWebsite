2. Description of Service

Shepherd is a macOS menu bar utility that monitors application windows for user-defined keywords and activity. The App reads text from windows using native macOS Accessibility APIs and AppleScript, running entirely on the user's local machine. Shepherd is designed for use cases such as parental supervision, self-monitoring, and productivity tracking.

Shepherd is offered in two tiers:

  • Free tier — Core local monitoring features, including keyword-based alerts and window watching.
  • Pro tier — All Free features plus Remote Supervision, which forwards alert metadata to a designated remote endpoint.

4. Accessibility Permissions

Shepherd requires macOS Accessibility permissions to function. You must manually grant these permissions in System Settings > Privacy & Security > Accessibility. This is a requirement imposed by Apple's operating system and is necessary for the App to read text content from application windows.

Shepherd will not function correctly without these permissions. Granting Accessibility access is entirely at your discretion, and you may revoke it at any time through System Settings. We recommend that you understand what Accessibility permissions allow before granting them.
